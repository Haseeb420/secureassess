# ngrok Setup

## Install
```bash
brew install ngrok/ngrok/ngrok
```

## Authenticate (one time)
```bash
ngrok config add-authtoken YOUR_TOKEN_HERE
```
Get token from: https://dashboard.ngrok.com/authtokens

## Get your free static domain (one time)
1. Go to https://dashboard.ngrok.com/domains
2. Click "+ New Domain"
3. ngrok assigns you a free static domain
   Example: `pleased-hedgehog-musical.ngrok-free.app`
4. Copy it — you will add it to `.env` files and `Makefile`

## Add to .env files

In `apps/api/.env`:
```
NGROK_STATIC_DOMAIN=pleased-hedgehog-musical.ngrok-free.app
```

In `apps/desktop/.env`:
```
VITE_API_BASE_URL=https://pleased-hedgehog-musical.ngrok-free.app
```

In `apps/admin/.env.local`:
```
NEXT_PUBLIC_API_BASE_URL=https://pleased-hedgehog-musical.ngrok-free.app
```
